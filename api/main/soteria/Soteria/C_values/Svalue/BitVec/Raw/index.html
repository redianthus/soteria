Module BitVec.Raw

Raw operations for values of type TBitVector; be careful when using these, so as to provide properly typed values. Prefer using BitVec directly when possible, as it may also provide more reductions.

val mk : int -> Z.t -> t
val mk_masked : int -> Z.t -> t
val mk_like : t_node Hc.hash_consed -> Z.t -> t
val bv_to_z : bool -> int -> Z.t -> Z.t

bv_to_z signed bits z parses a BitVector z, for a given bitwidth bits, with signed, into an integer.

val max_for : bool -> int -> Z.t

max_for signed n is the inclusive maximum for a bitvector of size n when it is signed

val min_for : bool -> int -> Z.t

min_for signed n is the inclusive minimum for a bitvector of size n when it is signed

val is_2pow : Z.t -> bool

is_2pow z is true if z is a power of 2 greater than one. The > 1 check is done to ensure the mask for that number (ie. z - 1) is nonzero.

val is_left_mask : int -> Z.t -> bool

is_left_mask bits z is true if z is of the form 1+0+ and its length is bits

val is_right_mask : Z.t -> bool

is_right_mask z is true if z is of the form 0*1+

val right_mask_size : Z.t -> int

right_mask_size z is, for a z of the form 0*1{n}, n. If z is not of the form 0*1{n}, the result is undefined, so use is_right_mask before.

val covers_bitwidth : int -> Z.t -> bool

covers_bitwidth bits z is true if z is of the form 1+ and covers the whole bitwidth of size bits.
